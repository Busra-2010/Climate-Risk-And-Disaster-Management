# Climate-Risk-And-Disaster-Management
This is an AICTE Internship.
This project is a machine learning-based system designed to classify disaster-related data. It helps in identifying and categorizing disaster information for better analysis and response.

Features: 
- Classifies disaster-related data
- Uses machine learning algorithms
- Interactive UI using Streamlit
- Real-time predictions

How It Works
1. Input disaster-related data
2. Data preprocessing and cleaning
3. Feature extraction
4. Apply classification model
5. Output predicted category
